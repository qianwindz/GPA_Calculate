# GPA & Grade Calculators

Free, privacy-friendly GPA and grade calculators for students.

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm lint`
- `pnpm test`
- `pnpm build`

## Project Rules

- Calculator formulas live in `src/lib/calculators/`.
- Interactive calculator UI lives in client components only.
- No database or server-side storage for grade inputs.
- Public pages should be static and SEO-friendly where possible.
